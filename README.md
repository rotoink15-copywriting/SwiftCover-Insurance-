# SwiftCover

SwiftCover is a modern, digital-first insurance website built to make insurance fast, clear, and trustworthy.  
The platform focuses on speed, simplicity, and strong visual credibility to convert visitors into confident policyholders.

---

## Overview

SwiftCover removes friction from insurance by combining clean UI, clear messaging, and realistic, premium visuals.  
Every section is intentionally designed to communicate trust within seconds.

---

## Key Features

- Conversion-focused landing page
- Auto-scrolling hero section with multiple visual frames
- Clear insurance coverage presentation (Auto & Health)
- Minimal, professional UI design
- Mobile-first and responsive layout
- Fast onboarding experience

---

## Visual Sections
